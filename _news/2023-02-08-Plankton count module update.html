A new version of Plankton Toolbox is now available.

The new version (1.4.0) of the plankton counting program Plankton Toolbox is available for both Windows and MacOS. The release includes an update of the counting module and several minor fixes. In addition the NOMP 2022 biovolume list, which includes information from the HELCOM-PEG 2022 biovolume list and additional taxa from the Skagerrak and the North Sea, are included.

The program can be downloaded at http://nordicmicroalgae.org/tools
